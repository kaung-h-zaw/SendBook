How SendBook works

  • Receivers create a short-lived session and get a code & QR.
  • Senders join with the 4-character code or by scanning QR.
  • One file at a time. Formats: .epub .mobi .azw .azw3 .pdf .txt
  • Files auto-delete after download, replacement, disconnect, or inactivity.
  • No accounts, no analytics — just minimal temporary storage.
